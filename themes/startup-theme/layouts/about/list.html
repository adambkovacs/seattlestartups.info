Why I Built This

This project was born from frustration. As a founder in Seattle, I spent weeks hunting down resources scattered across city websites, university portals, and buried LinkedIn posts. Every founder I talked to had the same problem: "Where do I even start?"

So I built what I needed: a single, verified, community-maintained directory of every free resource that actually helps early-stage founders. No pay-to-play listings, no affiliate links, just honest information.

This is a living project. If you find a resource I've missed or spot outdated information, please submit an issue or PR on GitHub. Together, we can make Seattle's startup ecosystem more accessible to everyone.
