## Accelerators and Incubators

### [AI2 Incubator](/resources/ai2-incubator/)

Born from the Allen Institute for AI (founded by Paul Allen), this is Seattle's premier AI incubator.

| Feature | Details |
|---------|---------|
| Funding | Up to $600,000 |
| Cloud credits | Up to $1M |
| Duration | 12 months |
| Check size | $100K - $500K (pre-seed) |
| Equity | Varies by deal |

**What's Included:**
- Access to AI researchers and engineers
- Office space
- Technical mentorship
- Investor network

**Notable Alumni:**
- Lexion ($35M Series B)
- Xembly ($20M+ raised)
- WellSaid Labs ($10M+ raised)
- KITT.AI (acquired by Baidu)

### [AWS Generative AI Accelerator](/resources/aws-activate/)

Amazon's flagship AI accelerator based in Seattle.

| Feature | Details |
|---------|---------|
| Duration | 8 weeks |
| AWS credits | Up to $1,000,000 |
| Format | Cohort-based |
| 2025 cohort | 40 startups |
| Kickoff | October 14, 2025 in Seattle |

**What's Included:**
- Personalized mentorship
- Technical architecture support
- Access to AWS AI/ML services
- Demo day with investors

### Techstars Seattle

Part of the global Techstars network with strong AI/ML cohorts.

| Feature | Details |
|---------|---------|
| Investment | $220,000 total |
| Structure | $20K for 5% + $200K uncapped SAFE |
| Network | 4,600+ companies, $116B market cap |
| Format | 3-month accelerator |

### Madrona Venture Labs

Venture studio specifically built for AI startups.

| Feature | Details |
|---------|---------|
| Format | 3-month venture studio |
| Focus | AI and ML startups |
| Requirements | Small team, early product, customer momentum |
| Connection | Madrona Venture Group |

### Pioneer Square Labs

Startup studio model with AI focus.

**How It Works:**
1. PSL creates and validates ideas internally
2. Recruits executive teams for validated concepts
3. Provides resources to launch
4. Stays involved as investor

### 9Mile Labs

B2B and enterprise software specialist with AI applications.

| Feature | Details |
|---------|---------|
| Focus | B2B and enterprise software |
| Specialty | Go-to-market strategy |
| Stage | Early-stage with product |
